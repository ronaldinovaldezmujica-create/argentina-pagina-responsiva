Informática

¿Qué es la informática?

La informática es la ciencia que estudia el tratamiento automático de la información mediante sistemas computacionales, incluyendo hardware, software y redes.

Hardware

El hardware es el conjunto de componentes físicos de una computadora, como la CPU, memoria, teclado y monitor.

Software

El software es el conjunto de programas que permiten que el hardware funcione y realice tareas específicas.
